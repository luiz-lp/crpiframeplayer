#### Instalação
- Chrome
    - Mais ferramentas > Extensões
    - Ativar modo programador
    - carregar expandida
    - selecione pasta Crunchyroll_Premium
- Kiwi Browser
    - Extensões
    - ativar modo desenvolvedor
    - load Crunchyroll_Premium_Kiwi-Browser
